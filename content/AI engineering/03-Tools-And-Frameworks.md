# AI Tools & Frameworks 🔧

*What you can use to work with AI - without coding!*

---

## What is a Framework?

Think of a framework like a **toolbox** 🧰

It gives you pre-made pieces so you don't have to build everything from scratch.

### Simple Analogy

Building with AI is like building with LEGO:

| Without Framework | With Framework |
|-------------------|-----------------|
| Gather raw plastic | Get LEGO bricks |
| Melt and mold | Stack and connect |
| Months of work | Hours of work |

---

## Popular AI Frameworks

### LangChain 🦜🔗

**What it is:** The most popular framework for building AI apps

**Best for:** Connecting AI to your data, databases, APIs

**Who uses it:** Developers building AI products

**Example Use:**
```
Your Question → LangChain → [Search Database] → AI → Answer
```

---

### LangGraph

**What it is:** LangChain's tool for building **agents** (AI that can think and act)

**Best for:** Complex multi-step AI workflows

**Think of it as:** Flowcharts for AI

---

### AutoGen

**What it is:** Microsoft's framework for building teams of AI agents

**Best for:** Multi-agent collaboration

**Example:**
```
Researcher Agent → Writer Agent → Editor Agent → Published Article
```

---

### CrewAI

**What it is:** A simpler way to create AI teams

**Best for:** Content creation teams

**Example:**
```
Researcher + Writer + SEO Expert = Blog Post Team
```

---

### Semantic Kernel

**What it is:** Microsoft's enterprise framework

**Best for:** Big companies integrating AI into existing products

## What is an Extension?

An extension adds **extra powers** to your AI:

| Base AI | + Extension | New Power |
|---------|-------------|-----------|
| ChatGPT | Code Interpreter | Runs code |
| Claude | Web Search | Current info |
| Any AI | PDF Reader | Reads documents |

---

## What is a Plugin?

A plugin is like an **app** for your AI:

- Adds specific features
- Can be turned on/off
- Usually created by other developers

### Examples

**ChatGPT Plugins:**
- Wolfram (math & science)
- Zapier (automation)
- Kayak (travel booking)

---

## What are Tools?

Tools are what AI uses to **do things** in the real world:

| Tool | What It Does |
|------|-------------|
| Web Search | Finds current information |
| Calculator | Does math accurately |
| Code Interpreter | Writes & runs code |
| File Reader | Reads documents |
| API Connector | Connects to other services |

## What is Claude Code?

**Claude Code** is a tool that lets you use Claude AI in your **terminal/command line**.

### What is a Terminal?
A text-based interface where you type commands (no mouse!)

### What Can It Do?
- Write code
- Edit files
- Run commands
- Navigate your computer

### Why Use It?
- Faster than GUI for developers
- Perfect for automation
- Great for repetitive tasks

---

## What is Ollama? 🦙

**Ollama** lets you run AI models **on your own computer** - for free!

### Why Use Ollama?

| With Ollama | Without Ollama |
|-------------|----------------|
| Free to use | Pay per use |
| Private (your data stays local) | Data sent to cloud |
| Works offline | Needs internet |
| Runs on your laptop | Requires cloud server |

### Models Available
- Llama 2 (Meta's model)
- Mistral
- CodeLlama
- And many more!

## When to Use What? 🤔

**Just want to chat?** → ChatGPT, Claude

**Want to build an AI app?** → LangChain

**Want AI to work on files locally?** → Ollama

**Want AI in your terminal?** → Claude Code

**Want to automate without coding?** → Zapier, Make

**Want AI agents to work together?** → CrewAI, AutoGen
