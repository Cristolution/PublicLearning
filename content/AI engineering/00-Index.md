## 📝 Quick Start Commands

### Install Ollama (Run AI locally)
```bash
# macOS/Linux
curl -fsSL https://ollama.com/install.sh | sh

# Then run a model
ollama run llama3
```

### Get an API Key
```
OpenAI: platform.openai.com → API Keys
Anthropic: console.anthropic.com → API Keys
Google: aistudio.google.com → API Keys
```
